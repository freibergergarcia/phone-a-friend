How it works

Three steps. One command.

Claude stays in the driver's seat. The backend does the heavy lifting. Results flow back into your session.

üí¨

You ask Claude

Describe what you need ‚Äî a code review, refactor, debug session, or architectural analysis.

üìû

Claude phones a friend

Your prompt, repo context, and optional diffs are relayed to Codex or Gemini in a sandboxed environment.

‚ú®

Get refined results

The backend's response flows back to Claude, who synthesizes it and presents the results in your session.

Two modes

Quick ask or deep collaboration.

One-shot for fast feedback. Multi-round for iterative refinement until convergence.

/phone-a-friend

One-shot relay

Ask once, get an answer. Perfect for code reviews, quick checks, and second opinions.

# Review recent changes
/phone-a-friend Review my
recent changes for issues
# Get a code review
/phone-a-friend Review the
error handling in relay.py
/phone-a-team

Iterative refinement

Multi-round loop with convergence checking. Claude delegates, reviews, and iterates up to 5 rounds.

# Run both backends
/phone-a-team --backend both
Refactor the backend registry
# Limit rounds
/phone-a-team --max-rounds 2
Redesign the auth module

Features

Built for safety and speed.

Guard rails, sandboxing, and zero dependencies. Production-ready from the first relay.

üîå

Backend-agnostic

Codex and Gemini today. Add new backends via the RelayBackend protocol ‚Äî swap or run both in parallel.

üîí

Sandboxed by default

Read-only sandbox prevents accidental writes. Escalate to workspace-write or full-access only when needed.

üì¶

Zero dependencies

Pure Python 3.10+. No pip install required. Clone the repo and start relaying immediately.

üîÑ

Multi-round refinement

The /phone-a-team loop delegates, reviews, and iterates with convergence checking across rounds.

üõ°Ô∏è

Depth guard

Recursive relay prevention via PHONE_A_FRIEND_DEPTH. No infinite agent loops ‚Äî fail-fast by design.

‚ö°

Size limits

Hardened limits on context (200 KB), diffs (300 KB), and prompts (500 KB) prevent oversized payloads.

Get started

Up and running in 60 seconds.

Clone, install the plugin, and start relaying.

  1. 1

    Clone the repository

    $ git clone https://github.com/freibergergarcia/phone-a-friend.git
    $ cd phone-a-friend
  2. 2

    Install the Claude Code plugin

    $ ./phone-a-friend install --claude
  3. 3

    Start relaying

    Use the slash commands in any Claude Code session.

    $ /phone-a-friend Review the auth module
